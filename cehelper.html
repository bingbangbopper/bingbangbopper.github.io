CE Helper

Corresponding "Array of byte":

Change the value type to "Array of byte" first, then paste the above string into CE, and hit First Scan.

Copy the addresses found to the address list below. (I found 2 in my scan just now, but for you might be different.) Use the above calculator to calculate a new "array of byte" string for your desired material counts (the max is 9999999). Then, paste this new "array of byte" string into the value for each address you moved below. (For me it works to click on an address, press Enter to select the value, paste the string to overwrite it, and press Enter to confirm.)

Finally, on the left side of each address is an empty white box; click each box to mark it with a red X. CE is now forcing your material counts to be your chosen material counts. Check the pause menu (or unpause and pause again) to see whether they changed.
